Datamax Prodigy Max label printer driver

I wrote this program for the Datamax Prodigy Max label printer at work that stopped working. Now it works WAY better than it ever did before. It prints MUCH faster, doesn't ever make any mistakes like it used to, and it's WAY easier to print now.

labels.zip - Download the whole project.
PRINTLAB.BAS - View the BASIC source.

LABELS-1 - An example file ordering two labels.

The Batch Files

Print One Label.BAT - Prints one of each label ordered.
Print Two Labels.BAT - Prints two of each label ordered
Restore Last Label Batch.BAT - Restores the last batch printed in case they need to be reprinted.
